Work Tracker Pro es una aplicación de gestión de trabajos y trabajadores diseñada para funcionar tanto en escritorio como en dispositivos móviles, incluso sin conexión a internet.

  • Versión escritorio: Aplicación Electron para Windows, macOS y Linux
  • Versión móvil: Progressive Web App (PWA) para dispositivos Android e iOS
  • Almacenamiento local: Guarda datos en IndexedDB para acceso offline
  • Dictado por voz: Permite dictar información en los formularios
  • Sincronización: Sincroniza datos cuando hay conexión a internet
